# MDvDM — Moderated Neutron Detection Studies

Simulations for optimizing moderated neutron detectors using scintillator coincidence techniques.

## Subprojects
- **Cf252ModeratorOptimization** — Optimize polyethylene moderator for Cf-252 fission neutrons.
- **LiF/LaBr3TaggingEfficiency** — Thermal neutron tagging efficiency with LiF + LaBr3(Ce).
- **LiF/ModeratedCf252LaBr3CeBr3Coincidence** — Full coincidence setup: Cf-252 source, moderator, lead collimator, LiF crystal, LaBr3 + CeBr3 detectors.
